Vim
===
My vimrc etc.

Prerequisites
-------------

Installation
------------
On the terminal do:
    $ git clone https://github.com/jbro/vim.git .vim
    $ curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
        https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
$ ln -s .vim/vimrc .vimrc

Start vim and do:
    :PlugInstall
