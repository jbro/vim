Vim
===
My vimrc etc.

Prerequisites
-------------
- Vim >= 7.3, with ruby support
- Development files for the ruby version vim is build with
- ctags

Installation
------------
    git clone --recursive git://github.com/jbro/vim.git ~/.vim && \
    sh ~/.vim/update && \
    ln -s ~/.vim/vimrc ~/.vimrc

Update
------
    sh ~/.vim/update.sh
